# AshitaUI2.0

A PyQt6-based UI for managing Ashita boot configs and startup scripts.

## Requirements

```
qtpy
PyQt6
requests
pygame
```

Install with:
```sh
pip install -r requirements.txt
```

## Features

- Edit Ashita boot configs
- Edit startup scripts

## Usage

Run the main entry point:
```sh
python -m src.main
```
